Plant Generator

Unity Plugin

Plant Generator enables developers
to create and grow 3D models
of plants directly in Unity
as well as merge two existing
plants into new ones.

Plant Generator is a Unity Plugin currently developed by Emil Wallraff and me in cooperation with CritCrew and is part of the course Game Lab II at Julius-Maximilians-Universit√§t.

The Plugin generates plants by creating a Lindenmayer system from the parameters set by the developer. In addition, the plugin can grow plants discreetly by increasing the iterations of the Lindenmayer system.

The generated plants consist of a trunk and branches, as well as leaves and flowers attached to the branches respectively. Furthermore spines can be added to the branches.

In addition, the plugin can generate and grow multiple plants at the same time in a set area. Also, developers can import custom 3D meshes and use them to replace the standard leaves, flowers and spines.

With the plugin you can save your plants by generating a seedcode file and regenerate or grow plants out of these seedcode files. Furthermore, the plugin lets the developers merge two plants using their seedcodes by merging the individual parameters at the request of the developer in each case.
