You only live once! is a game I developed using the Unity Engine during my first year at Julius-Maximilians-Universit√§t as part of the course Game Lab I. It is a dynamic 2D top-down shooter with bullet hell elements.

The game features a series of epic boss fights, in which the player tries to achieve the highest score. Each run is infinite, however the difficulty is increasing with each completed boss fight.

It includes two different bosses with unique movement and shooting patterns depending on their difficulty level. Other features include background music and sound effects, as well as various particle effects and animations of the player and opponents.

The video game includes an interactive tutorial in which the player can playfully learn the controls and the game mechanics.

After each completed boss fight the player can either choose an item which either reduces or heightens the difficulty of the current run. However, fewer points are awarded, if the difficulty decreases and vice versa.

Many assets from You only live once! were created by myself including all menus. The main menu lets the player access all important menus and shows the current highest score of the player which is stored between different sessions.

The player can view all items in the unlockables menu. It shows all unlocked items with their respective changes as well as all locked items and which score is needed to achieve to unlock them.

The player can play the game by using either keyboard and mouse or a gamepad (tested with XBox Wireless Controller and Nintendo Switch Pro Controller).

The game lets the player freely assign the keys of the keyboard and mouse, as well as the gamepad. The custom key binds are stored between multiple sessions.
